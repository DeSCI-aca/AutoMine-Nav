📦 AutoMine-Nav Dataset

🔍 Dataset Overview

AutoMine-Nav is a comprehensive multimodal dataset collected from open-pit mining areas, designed for autonomous vehicle localization and navigation tasks.

The data collection platforms include:

  • One SUV
  • One Wide-Body Truck
  • One Mining Truck

The SUV platform records synchronized data from:

  • Ouster OS1-64 LiDAR
  • Stereo Cameras (Left & Right)
  • GNSS/INS System
  • IMU

📌 Extrinsic Parameters

All sensor-to-sensor transformation parameters are provided in the docs/date_vehicle_type/ directory, including:

  • param.json: LiDAR-to-Camera (Left & Right), IMU-to-LiDAR, GPS-to-IMU
  • ROS_Left_Right.txt: Stereo camera baseline information

Coordinate Convention: ROS standard (Right-handed) - X-forward, Y-left, Z-up.
