üîç Dataset Introduction

AutoMine-Nav is a multimodal dataset suitable for autonomous driving localization tasks. The data was collected from multiple mining areas, including:

  • Ouster-I-64 LiDAR
  • Left Camera
  • Right Camera
  • GPSS
  • IMU

It can be widely used for algorithms such as Odometry and SLAM.

üìå Extrinsic Parameters Information

The dataset provides detailed extrinsic parameters between all sensors:

  • LiDAR-to-Camera: Transformation matrices for both left and right cameras
  • IMU-to-LiDAR: Precise calibration parameters
  • GPS-to-IMU: Lever arm offset and orientation

All extrinsic parameters are provided in the calibration folder of each sequence, including:

  • JSON files with transformation matrices
  • Detailed documentation on coordinate systems
  • Visualization tools for verification

The coordinate system follows ROS convention (right-handed, X-forward, Y-left, Z-up).
